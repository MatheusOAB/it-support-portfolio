# User Login Issue Simulation

## Objective
Simulate and resolve a user login issue by managing local user accounts and passwords.

## Tools
- Windows
- PowerShell

## What I did
- Checked existing users using PowerShell
- Created a new local user
- Assigned a password to the user
- Simulated a login issue by not knowing the user's password
- Reset the user password

## Issues & Fixes
- Didn't remember how to list users → used `Get-LocalUser`  
- Forgot the user password → reset password using PowerShell  
- Verified changes by rechecking user access  

## Outcome
Successfully created a user, simulated a login issue, and resolved it by resetting the password.

## What I learned
- How to list local users in PowerShell  
- How to create and manage user accounts  
- How to reset passwords  
- Basic troubleshooting for login issues
